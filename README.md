# CookUnity Sr. Full Stack challange

## Getting started

Backend

```bash
cd /back
npm install
npm run dev
```

Frontend

```bash
cd /front
npm install
npm run dev
```

## Instructions

Required

- Make the tests passes.

```bash
# start back and front dev servers then
cd /front
npm run tests
```

Optional

- Feel free to improve/refactor/add/remove anything you like in the existent code.
